# Privacy Policy

## GDPR & Data Protection Statement

This Robot Management System collects limited Personally Identifiable Information (PII) as part of its user authentication and mission logging features. The system stores usernames, encrypted passwords, user roles (Viewer or Commander), and timestamps of user actions. When a user sends a command to the robot, the system logs the username, command type, timestamp, and related robot status data. No additional personal information is collected, in accordance with the principle of Data Minimisation under GDPR.

The purpose of collecting this information is to ensure accountability and safety. Mission logging provides an audit trail of who sent each command to the robot and when it was sent. This supports system monitoring, debugging, and meeting the safety requirements of the Ground Control Station.

All data is stored securely within the application’s local database. Passwords are encrypted and never stored in plain text. Access to audit logs is restricted based on user roles. In line with the principle of Storage Limitation, data will only be kept for as long as necessary to support the academic assessment and system evaluation purposes of this project.
